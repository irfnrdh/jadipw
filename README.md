## 🎯 Tentang

Self-Hacking adalah panduan dan toolkit komprehensif untuk optimasi diri. Repository ini berisi semua yang kamu butuhkan untuk menganalisis, memahami, dan mengoptimalkan sistem personal - mulai dari kebiasaan dan pola pikir hingga rutinitas harian dan workflow produktivitas.

### Apa itu Self-Hacking?

Self-hacking adalah proses meningkatkan performa personal secara sistematis dengan:
- 📊 **Melacak** aktivitas dan pola harian
- 🔍 **Menganalisis** apa yang berhasil dan tidak
- ⚡ **Mengoptimalkan** sistem untuk hasil yang lebih baik
- 🔄 **Beriterasi** berdasarkan data dan feedback

## 🚀 Mulai Cepat

### 1. Pilih Area Focus Kamu
```bash
# Clone repository
git clone https://github.com/irfnrdh/self-hacking.git
cd self-hacking

# Mulai dengan satu area
├── produktivitas/   # Manajemen waktu dan fokus
├── kesehatan/       # Wellness fisik dan mental
├── kebiasaan/       # Pembentukan dan penghilangan kebiasaan
└── tools/           # Template tracking dan aplikasi
```

### 2. Mulai Tracking
Gunakan template kami untuk mulai mengumpulkan data:
- [Lembar Tracking Harian](template/tracker-harian.md)
- [Template Review Mingguan](template/review-mingguan.md)
- [Checklist Audit Bulanan](template/audit-bulanan.md)

### 3. Implementasi & Iterasi
Ikuti [Tantangan 30 Hari](tantangan/) untuk membangun momentum dan melihat hasil.
